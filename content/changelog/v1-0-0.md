First release of Boiler Room. Built on Next.js with content stored as Markdown files

--
This release includes feature updates:
- **Writing** — a section for essays and longer posts
- **Bookshelf** — book notes grouped by reading status (to-read, reading, finished, paused)
- **Notes** — short fragments with pinned entries and kind categories

--
Tech Stack:
-  **NextJS** — Flexible frontend/backend for RSS
- **CMS** — Sveltia CMS wired to GitHub
- **Vercel** — continuous deployment
